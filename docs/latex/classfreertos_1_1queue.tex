\subsubsection*{template$<$size\+\_\+t Queue\+Length, typename T, typename Queue\+Allocator$>$\newline
class freertos\+::queue$<$ Queue\+Length, T, Queue\+Allocator $>$}
A modern C++ wrapper for Free\+RTOS queues with type safety. 

This class provides a type-\/safe, RAII wrapper around Free\+RTOS queues for inter-\/task communication. It supports compile-\/time type checking, automatic resource management, and std\+::chrono timeout support.

\hline

\hline
