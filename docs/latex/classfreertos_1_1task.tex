\subsubsection*{template$<$typename Task\+Allocator$>$\newline
class freertos\+::task$<$ Task\+Allocator $>$}
A modern C++ wrapper for Free\+RTOS tasks with RAII semantics. 

This class provides a type-\/safe, exception-\/safe wrapper around Free\+RTOS tasks. It automatically manages task lifecycle and supports both static and dynamic memory allocation strategies.
