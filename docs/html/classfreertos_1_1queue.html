Detailed Description

template<size_t QueueLength, typename T, typename QueueAllocator>
class freertos::queue< QueueLength, T, QueueAllocator >

A wrapper for the FreeRTOS queue.

Template Parameters
QueueLengthThe maximum number of items that the queue can hold.
TThe type of the items that the queue will hold.
QueueAllocatorThe type of the allocator to use.
