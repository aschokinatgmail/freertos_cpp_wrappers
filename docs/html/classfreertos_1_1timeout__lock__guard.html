Detailed Description

template<typename Mutex>
class freertos::timeout_lock_guard< Mutex >

Lock guard for the mutex with timeout. The mutex is locked in the constructor and unlocked in the destructor (RAII).

Template Parameters
Mutextype of the mutex to guard.
